For memory validation and testing, Smart Phone Flash Tool offers a Memory Test for users to test and verify the external memory devices RAM, eMMC and NAND flash.  In the test mode, the DA is downloaded into the target¡¯s internal SRAM to check the handset¡¯s memory devices. Smart Phone Flash Tool will parse the memory deceive configuration information from the Pre-Loader.

 

l  Memory Test steps

 

1.       Switch to Memory Test

 

To enter the Test Mode, select the ¡°Memory Test¡± page.

 

 

2.     Select Memory Test Methods

 

For external RAM testing, we provide four test methods: Data Bus Test, Address Bus Test, Decided Pattern Test, Inc/Dec Pattern Test; For the NAND or eMMC flash test, we provide a test method, Decided Pattern Test. You can choose the test scenarios from the left check-boxes.

 

 

 

3.     Power off target

 

A handshake must be performed between Smart Phone Flash Tool and the target mobile phone, thus the target must initially be powered off, and connected to the PC via a download cable.

For USB download, please pull out both Cable and Battery.

 

4.       Select DA

 

Please make sure the Download Agent has been assigned.

 

 

5.       Select Scatter File

 

Please make sure correct scatter file has been assigned.

 

 

6.       Configure Options

 

Open ¡°Option Dialog¡± to select the expected options for download method, power supply and speed switch mode.

 

7.       Start Memory Test

 

Press ¡°Start¡± button in Memory Test page to start downloading image.

 

 

8.       Power on target

 

Power on the target and the format procedure will begin.

 

9.       Stop Memory Test

 

Note that to interrupt the format process at any time, press the stop button or f10.
