l  Readback Options

 

1.       Assign Readback Files

Click the§Add§ button to add a read back file, and ※Remove§ button will remove the selected item.

 

 

2.       Configure Readback Options

Double click the readback item to assign expected file path and name.  After click ※Save§ button, following options should be configured.

 

 

l  Select Read Flag (Only for NAND Flash)

每         Read Page + Spare: To read NAND flash page and spare raw data including bad blocks;

每         Read Page Only: To read NAND flash page raw data including bad blocks;

每         Read Spare Only: To read NAND flash spare raw data including bad blocks;

每         Read Page with ECC: To read NAND flash good block page data with ECC 1-bit correction mechanism

 

l  Configure Address Type

每         Hex: value is used immediately.

每         Decimal: value is automatically converted into hexadecimal in the main application window.

 

l  Configure Start Address and Length

每         NOTE: The length must be an integer multiple of 4096.

 

 

n  Readback Step

 

l  Power off target

A handshake must be performed between Smart Phone Flash Tool and the target mobile phone, thus the target must initially be powered off, and connected to the PC via a download cable.

For USB download, please pull out both Cable and Battery.

 

l  Select DA

Please make sure the Download Agent has been assigned.

 

 

l  Select Scatter File

Please make sure correct scatter file has been assigned.

 

l  Assign readback file

Add readback items by pressing ※Add§ button.

 

l  Configure Options

Double click the readback item to configure file path, file name, address type, read flag,  start address and length.

        

l  Start Readback

Press ※Readback§ button to start downloading image.

 

 

l  Power on target

Power on the target and the readback procedure will begin.

 

l  Stop Readback

Note that to interrupt the readback process at any time, press the stop button or f10.
