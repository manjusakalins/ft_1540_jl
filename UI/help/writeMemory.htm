Flash Tool provides Write Memory function for users to write specified file to specified address on the flash.

 

l  Enter Write Memory Page

 

Write Memory Page can be shown by pressing ¡°Window->Write Memory¡±.

 

 

l  Write Memory Steps

 

1.       Power off target

 

A handshake must be performed between Smart Phone Flash Tool and the target mobile phone, thus the target must initially be powered off, and connected to the PC via a download cable.

For USB download, please pull out both Cable and Battery.

 

2.       Select DA

 

Please make sure the Download Agent has been assigned.

 

 

3.       Select Scatter File

 

Please make sure correct scatter file has been assigned.

 

4.       Select file to write

 

Switch to write memory page, select specific file to write.

 

5.       Configure begin address

 

Configure specific address from where the file will be written.

 

6.       Start write memory

 

Press ¡°Write Memory¡± button to start downloading image.

 

7.       Power on target

 

Power on the target and the download procedure will begin.

 

8.       Stop write memory

 

Note that to interrupt the download process at any time, press the stop button or f10.
