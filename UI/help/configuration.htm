Ø  Select Scatter File

 

Please select the scatter file first according the chip which is supposed to execute. By scatter file, SP Flash Tool will be notified what type Smart Phone     

chip will be operated. For instance, if the platform is MT6589, please select MT6589_Android_scatter.txt first.

 

Scatter File could be loaded by ¡°Scatter-loading¡± button or select in history list, and the file can be found in the load folder.

 

 

Ø  Assign Images

 

If scatter file has been loaded, then all image files in the load folder will be automatically loaded.

To manually a special image file for certain ROM, click on the ROM location: a file browser will pop up for file selection. 

 

 

 

Ø  Select Download Scene

 

Please select expected download scene before download, firmware upgrade is the default scene.

There are four scenes for download operation: Like MOTA, Format All + Download, Firmware Upgrade and Custom.

ROMs Selection in Like MOTA, Format All + Download, Firmware Upgrade is unchangeable. Scene selection will change to Custom automatically if any ROM selection change is detected.
